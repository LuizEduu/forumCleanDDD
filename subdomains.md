# Subdomínios

- Core: O que dá dinheiro / não pode parar
- Supporting: dá suporte para o core funcionar
- Generic: Você precisa deles, mas não são tão importantes

## Exemplos

### core:

- Compra
- Catálogo
- Pagamento
- Entrega

### supporting

- Estoque

### Generic

- Notificação ao cliente
- Envio de e-mail
- Promoções
- Chat
